# 浏览器示例

功能：

- 连接房间实时显示弹幕/礼物，可勾选屏蔽的消息类型
- **录制**：连接前勾选"录制弹幕"，弹幕会缓存在内存里（屏蔽选项不影响录制内容），随时可下载为 `.jsonl` 文件——格式与 CLI `douyudm -i 9999 --record` 的输出完全相同
- **转换**：页面底部"JSONL → ASS 字幕转换"区，选择录制文件（本页下载的或 CLI 生成的都行），可设置起止时间、延迟、剔除正则、字号、滚动秒数，转换在浏览器本地完成，产出 `.ass` 字幕文件

注意：录制数据在内存中，刷新或关闭页面就没了，长时间录制建议用 CLI。

## 快速启动

直接用浏览器打开 `index.html` 即可，无需安装任何依赖（通过 CDN 加载）。

```bash
open index.html
```

或者启动一个本地服务器（避免部分浏览器的跨域限制）：

```bash
# Python
python3 -m http.server 8080

# Node.js
npx serve .
```

然后访问 `http://localhost:8080`。
