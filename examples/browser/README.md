# 浏览器示例

功能：

- 连接房间实时显示弹幕/礼物，可勾选屏蔽的消息类型
- **录制**：连接前勾选"录制弹幕"，弹幕分批写入浏览器本地数据库 IndexedDB（内存里只留几十条的小缓冲，长时间录制不会越积越多；屏蔽选项不影响录制内容）。录制中可随时下载，刷新或关闭页面后数据仍在，可在"历史录制"列表里下载或删除。下载的 `.jsonl` 与 CLI `douyudm -i 9999 --record` 的输出格式完全相同
- **转换**：页面底部"JSONL → ASS 字幕转换"区，选择录制文件（本页下载的或 CLI 生成的都行），可设置起止时间、延迟、剔除正则、字号、滚动秒数，转换在浏览器本地完成，产出 `.ass` 字幕文件

注意：IndexedDB 归属"当前网址 + 当前浏览器"，换浏览器或清站点数据会清掉历史录制；要长期保存请下载文件。

## 快速启动

直接用浏览器打开 `index.html` 即可，无需安装任何依赖（通过 CDN 加载）。

```bash
open index.html
```

或者启动一个本地服务器（避免部分浏览器的跨域限制）：

```bash
# Python
python3 -m http.server 8080

# Node.js
npx serve .
```

然后访问 `http://localhost:8080`。
